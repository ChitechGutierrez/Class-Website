Evernote and The Evernote Web Clipper

Possibly the most useful tool for research... ever

What is Evernote?

"As one workspace that lives across your phone, tablet, and computer, Evernote is the place you write free from distraction, collect information, find what you need, and present your ideas to the world. Whatever you’re working toward, Evernote’s job is to make sure you get there." - Evernote Website

In other words, it is a note taking tool that exists in the cloud, so it can be accessed from almost anywhere.

In Evernote, you can have various "Notebooks" for different topics, or you can put everything in just one giant Inbox notebook. Each "Page" can be used for different topics.

One of the most useful features is that you can add tags to your notes. These tags allow for easy searching about whatever topic you like, so it requires very little effort to find all your notes related to what you need.

Note types include:

  • Written
  • Pictures
  • Spoken Word
  • Lists
  • Reminders

    • The thing that makes Evernote truly the most powerful tool you can have when doing research, or even just exploring the internet, is the Evernote Web Clipper.

    This Chrome addon gives you the ability to save any web page, or article you find into your Evernote account. So once you find something useful, you hit a button, add a few tags and save it for later use. It automatically includes all source information as well, so it can later be cited or returned to.
